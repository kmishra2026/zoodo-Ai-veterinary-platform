# 🐾 Zoodo – AI & Blockchain Veterinary Platform

A modern pet healthcare platform that leverages AI and blockchain to provide intelligent care assistance, secure medical records, and seamless connectivity between pet owners and service providers.

🔗 Live Demo: https://zoodo.dev  
⚠️ Currently under active development

---

## 🚀 Key Highlights

- AI-driven pet care assistant for symptom analysis and recommendations  
- Secure digital health records powered by blockchain  
- Smart provider discovery using location and service intelligence  
- Integrated video consultation and online appointment system  
- End-to-end platform supporting pet owners, veterinarians, and trainers  

---

## 🧠 Core Features

### AI-Powered Assistance
- Symptom analysis with urgency detection  
- Personalized care and diet recommendations  
- Conversational AI assistant for pet health queries  
- Multi-modal support (text, image, speech)

### Blockchain-Based Records
- Tamper-proof medical records using smart contracts  
- Cryptographic verification and audit trail  
- Hybrid storage (database + blockchain for integrity)

### Platform Capabilities
- Appointment scheduling (online, in-person, home visits)  
- Geolocation-based provider matching  
- Training and behavioral services integration  

### Additional Integrations
- Secure payments via Razorpay  
- Video consultations using Jitsi Meet  

---

## 🛠 Tech Stack

- **Frontend:** Next.js, TypeScript  
- **Backend:** Spring Boot  
- **AI Service:** FastAPI (Python)  
- **Database:** PostgreSQL, MongoDB, Redis  
- **Blockchain:** Solidity, Hardhat, Polygon  

---

## 🏗 Architecture Overview

The system follows a microservices architecture:

- Frontend handles user interaction  
- Backend manages business logic and APIs  
- AI service processes intelligent recommendations  
- Blockchain ensures data integrity and verification  
- Databases support structured and unstructured data
